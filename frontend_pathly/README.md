# Pathly

An AI-powered learning roadmap generator built with React and TypeScript.

## What it does

Enter any topic and get a structured learning path with topics, descriptions, and resources. The roadmap is interactive with collapsible sections. You can also download roadmaps as JSON or browse previously generated ones.
